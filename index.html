A light-weight CQRS framework

This project attempts to provide a framework for using the CQRS pattern. It borrows heavily from other CQRS examples/implementations out there with an attempt to simplify the usage, and enable a rapid start for new projects using it.

Getting Started

var setup = new SetupClassThatReplacesIoCContainerFramework()
    .WithEventStore((bus, converter, conString) => 
        new DocumentEventStore(bus, converter,
            new FileDocumentStore(folderPath, new JsonDocumentStrategy())))
    .WithReadStorage(new FileDocumentStore(folderPath, new JsonViewStrategy()))
    .WithEventConverter(new EventConverter())
    .AssembleComponents();

ServiceLocator.Configure(setup);
